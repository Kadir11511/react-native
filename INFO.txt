Proje Özeti:
Kullanıcıların mobil cihaz üzerinden güvenli, hızlı ve kolay bir şekilde araç kiralama işlemi yapabilmesini sağlar.
Giriş ekranı güvenli hale getirilmiş, admin ve kullanıcı giriş yetkileri ayrılmıştır.
Kullanıcı araçları listeleyebilir, detay görebilir, kiralama formu doldurabilir.
Admin panelinden kiralama talepleri yönetilir.

Kullanılan Teknolojiler:
- Frontend: React Native
- Backend: Firebase Authentication ve Firestore DB
- API: Firebase Functions
- Bildirim: Firebase Cloud Messaging

Veritabanı:
- Users: Kullanıcı bilgileri
- Admins: Admin bilgileri
- Cars: Araç listesi
- Reservations: Kiralama talepleri
- Favorites: Favori araçlar

Kimlik doğrulama Firebase Authentication ile yapılmaktadır. Yetkisiz giriş engellenmiştir.
